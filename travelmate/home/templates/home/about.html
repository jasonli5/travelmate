About

TravelMate is a web application that makes planning and packing for trips easy. It provides users with useful tools such as weather-appropriate outfit recommendations, travel tips, and a packing list.

TravelMate uses real-time weather data to give accurate outfit suggestions. Users can also create and save “trips” with destination and packing details. These trips can be accessed from any device and shared with others to streamline the process.
