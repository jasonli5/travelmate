About

TravelMate is a web application that makes planning and packing for trips easy. TravelMate provides users with many useful tools such as weather appropriate outfit recommendations, travel tips, and a packing list.

TravelMate can accurately give outfit suggestions because it uses real-time weather data. This web application also allows users to be able to make specific “trips” that can be saved and updated, provided they have created an account. These trips store the destinations information as well as any packing lists curated. TravelMate can also be accessed from any devices and the packing lists can be shared with others to simplify the packing process for others on the trip. Users can make an account on the app as well.
